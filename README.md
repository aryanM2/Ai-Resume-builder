# ResumeAI Builder

ResumeAI Builder is a full-stack web application that helps users create professional resumes, improve them with AI, and check how well they match a job description using an ATS score analyzer.

## Features

* Create and manage multiple resumes
* AI-powered resume enhancement
* ATS score analysis based on job descriptions
* Multiple resume templates
* Download resumes as PDF
* Secure user authentication
* Razorpay payment integration
* Responsive design for desktop and mobile

## Tech Stack

**Frontend**

* React.js
* Tailwind CSS
* Redux Toolkit

**Backend**

* Node.js
* Express.js
* MongoDB
* JWT Authentication

**Other Tools**

* OpenAI / Gemini API
* Cloudinary
* Razorpay

## Installation

Clone the repository

```bash
git clone https://github.com/aryanM2/ResumeAI-Builder.git
cd ResumeAI-Builder
```

Install dependencies

```bash
# Frontend
cd client
npm install

# Backend
cd ../server
npm install
```

Create `.env` files in both the `client` and `server` folders and add the required environment variables.

Run the project

```bash
# Backend
cd server
npm run dev

# Frontend
cd client
npm run dev
```
